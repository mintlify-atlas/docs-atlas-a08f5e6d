## Key Features

### Rich Text Editor
Create posts with a powerful editor supporting formatting, headers, lists, and more. Export your content in JSON format for portability.

### User Authentication
Secure authentication using Lucia with Argon2 password hashing ensures your account stays protected.

### Profile Customization
Personalize your profile with:
- Custom username and bio (up to 140 characters)
- Profile picture with built-in cropping tools
- View all your published posts in one place

### Engagement System
Connect with the community through a like system that lets you save and appreciate posts from other users.

<Note>
Esoteric Network focuses on meaningful content creation. All posts are public and can be discovered by the community.
</Note>

## Technology Stack

Esoteric Network is built with modern web technologies:

- **Framework**: Next.js with App Router
- **Authentication**: Lucia with Argon2 hashing
- **Database**: PostgreSQL with Drizzle ORM
- **Editor**: Tiptap rich text editor
- **UI**: React with TypeScript
